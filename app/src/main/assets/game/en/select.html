Choice of the planet

Here the planets available to a repeated game are shown. Availability of this window is possible only if there is at least one freed planet.

Each planet is presented in the form of the ribbon in which elements, the preview and the name are displayed. If the planet is inaccessible for start, her preview is blocked by

In this case it is considered that the planet isn't freed yet. For a planet unblocking, it is necessary in the mode game to release it and then it will become available in the ribbon.
