Достижения

Здесь показываются завершенные сеансы игры, если нет ни одного - окно блокируется. Информация представлена в виде списка, в каждом элементе которого, отображается статистика в следующих категориях:
Убитых желтых существ, больше лучше.
Убитых красных существ, больше лучше.
Убитых зеленых существ, больше лучше.
Собранных яиц, больше лучше.
Потеряных дроидов, меньше лучше.
Общее время игры, меньше лучше.
Потраченных бомб, меньше лучше.
Потраченное топливо, меньше лучше.
Накопленный опыт, больше лучше.
Освобожденных планет, больше лучше.
Игровых циклов, меньше лучше.
Каждая категория показана в пропорции, относительно максимальных значений по каждой из них.
Также на элементе списка показана дата игры и название последней планеты.
Показывают общий рейтинг записанной игры.
Возможность просмотреть запись этой игры.
