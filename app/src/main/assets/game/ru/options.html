Настройки

Основные настройки

В основной области окна представлены переключатели и ползунки для изменения режимов:
  • Фоновая музыка и в ползунке выставляется громкость.
  • Звуки событий и в ползунке выставляется громкость.
  • Масштабирование используется для изменения размеров элементов в игровом экране(больше/меньше). Это бывает полезно, если вам не подходит размер выбранный программой для вашего девайса.
  • Скорость позволяет изменять(ускорять/тормозить) игровой процесс. Возможно, что скорость работы игры, которая выбрала программа для вашего устройства, вас не удовлетворил. Тогда можно смело менять эту настройку.
  • Классика при активации этого режима, графика игры переключается в классический стиль(ZX Spectrum 1987 год)
  • Мастер влияет на игровой процесс. При активации игра становится более сложной. Инвентарь более скудный. Для продвинутых игроков.
  • Автор влияет на игровой процесс. При активации персонаж игрока становится неуязвимым(режим бога). Но этот режим становится доступен только при определенных условиях, см.

Возможности

  1. Можно поменять основную тему игры. В игре 2 темы - светлая/темная. По умолчанию выставлена - темная. Выберите себе по вкусу.
  2. Это влияет на игровой процесс, надо выбрать систему планет, которя будет использована игрой по умолчанию. При установке игры, у вас будет только одна система - CLASSIC. При необходимости можно выбрать другую, создать свою в , либо загрузить с сервера обновлений. Для загрузки с сервера необходимо, в списке систем, выбрать "..." и в появившейся форме выбрать систему.
